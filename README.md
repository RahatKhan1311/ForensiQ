# ForensiQ - AI Crime Case Analyzer

ForensiQ is a web-based application designed to help law enforcement, investigators, and analysts manage and analyze crime cases efficiently. The system combines case management, report generation, and AI-assisted analysis in a single intuitive dashboard.

---

## 🚀 Features

### Case Management
- View all cases in a dashboard.
- Select or create new crime cases.
- Quick navigation between cases.

### Reports
- Generate PDF summaries of selected cases.
- Download and store case reports for offline use.

### Settings & Preferences
- **Profile Management:** Update name, email, and password.
- **Themes:** Dark mode and neon accents, with live preview.
- **Sidebar Behavior:** Sticky or collapsible.
- **Auto-refresh Interval:** Set dashboard refresh time.
- **API Integrations:** Connect OCR and AI analysis services.
- **Advanced Options:** Enable debug mode or reset app data.

### User Experience
- Fully responsive dashboard.
- Persistent user preferences using `localStorage`.
- Intuitive UI using Tailwind CSS.

---

## 🛠 Technologies Used
- **Frontend:** HTML, CSS (Tailwind), JavaScript
- **Storage:** Browser `localStorage` for settings
- **Backend (Optional):** APIs for case and report management
- **PDF Reports:** Generated via fetch requests to backend endpoints
